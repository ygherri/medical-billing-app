# Medical Billing App

A fullstack medical billing application built with Angular, NestJS, Prisma and PostgreSQL.

## Project goal

The goal of this project is to manage daily medical billing records for a small medical office.

## Planned features

- Manage patients
- Create daily invoices
- Select payment methods
- Select coverage types
- Filter invoices by date
- Display daily revenue summary
- Update and delete invoices

## Tech stack

- Angular
- NestJS
- TypeScript
- Prisma
- PostgreSQL
- REST API
